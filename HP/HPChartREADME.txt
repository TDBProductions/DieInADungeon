This is the file read in to determine character HP.  It is listed out by class, bonus and the number of dice to roll.

Class|BonusDiceRoll|NumberOfDiesToRoll|Diceroll

For example:
	> Ranger|8|1|8
This implies that a ranger rolls 1d8 for a base HP, then 1d8 for every level gained.
	> Wizard|0|1|4
This implies that a wizard rolls 0 for a base HP, then 1d4 for every level gained.
