The files in this folder are read into the program in order to find the character resistances.  The files are in the following format:

THAC0Lvl1|THAC0Lvl2|THAC0Lvl3|THAC0Lvl4|THAC0Lvl5
PoisonResLvl1|PoisonResLvl2|PoisonResLvl3|PoisonResLevel4|PoisonResLevel5
PetrifyResLvl1|PetrifyResLvl2|PetrifyResLvl3|PetrifyResLvl4|PetrifyResLvl5
WandResLvl1|WandResLvl2|WandResLvl3|WandResLvl4|WandResLvl5
BreathResLvl1|BreathResLvl2|BreathResLvl3|BreathResLvl4|BreathResLvl5
SpellResLvl1|SpellResLvl2|SpellResLvl3|SpellResLvl4|SpellResLvl5

Editting these files will edit the resistances for the associated class.  The default values have been taken directly from the old school DND books.
